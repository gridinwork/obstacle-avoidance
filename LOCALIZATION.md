# Локализация интерфейса

## Описание

Интерфейс программы поддерживает русский и английский языки. По умолчанию установлен русский язык.

## Использование

### Автоматический выбор языка

По умолчанию программа запускается с русским интерфейсом. Все строки интерфейса автоматически переводятся.

### Переключение языка программно

Если нужно изменить язык в коде:

```python
from gui.i18n import set_language, get_language

# Установить русский язык
set_language("ru")

# Установить английский язык
set_language("en")

# Получить текущий язык
current_lang = get_language()
```

### Добавление новых переводов

Для добавления новых строк в файл `gui/i18n.py`:

1. Добавьте строку в словарь `TRANSLATIONS["ru"]`:

```python
TRANSLATIONS = {
    "ru": {
        "Your new text": "Ваш новый текст",
        # ...
    }
}
```

2. Используйте функцию `tr()` в коде:

```python
from gui.i18n import tr

label.setText(tr("Your new text"))
```

### Форматирование строк

Для строк с параметрами используйте форматирование:

```python
# В i18n.py
"Opening camera ({device})...": "Открытие камеры ({device})...",

# В коде
tr("Opening camera ({device})...", device="USB0")
```

## Переведенные элементы

- ✅ Все кнопки и метки
- ✅ Названия вкладок
- ✅ Сообщения статуса
- ✅ Названия групп настроек
- ✅ Сообщения об ошибках
- ✅ Сообщения камеры и LiDAR
- ✅ Все текстовые элементы интерфейса

## Технические детали

Система локализации использует простой словарь переводов без внешних зависимостей. Функция `tr()` автоматически выбирает перевод на основе текущего языка или возвращает оригинальный текст, если перевод не найден.

## Изменение языка по умолчанию

Чтобы изменить язык по умолчанию, отредактируйте файл `gui/i18n.py`:

```python
# Изменить язык по умолчанию на английский
_current_language = "en"
```

Или установите язык при запуске программы в `main.py`:

```python
from gui.i18n import set_language

# Установить язык перед запуском GUI
set_language("ru")  # или "en"
```
