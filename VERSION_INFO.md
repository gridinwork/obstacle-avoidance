# Информация о версиях

## Версия 74-RU (Русский интерфейс)

**Текущая версия** - Полностью переведенный интерфейс на русский язык.

### Особенности:
- ✅ Полный русский интерфейс
- ✅ Все кнопки, метки и сообщения переведены
- ✅ Поддержка переключения языка (русский/английский)
- ✅ Система локализации для будущих языков

### Как использовать:
- По умолчанию интерфейс на русском языке
- Для переключения на английский используйте `set_language("en")` в коде
- См. `LOCALIZATION.md` для подробностей

---

## Версия 74-EN (Английский интерфейс)

**Предыдущая версия** - Интерфейс на английском языке.

### Как получить английскую версию:
1. Используйте тег `v74-EN` в Git
2. Или установите язык программно: `set_language("en")`

---

## Переключение между версиями

### Через Git:
```bash
# Русская версия (текущая)
git checkout main

# Английская версия (тег)
git checkout v74-EN
```

### Программно:
```python
from gui.i18n import set_language

# Русский интерфейс
set_language("ru")

# Английский интерфейс
set_language("en")
```
